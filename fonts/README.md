Place official DM Sans and DM Mono woff2 files here to self-host fonts.

Recommended filenames:

- DM-Sans.woff2
- DM-Mono.woff2

How to obtain (manual):

1. Go to https://fonts.google.com/ and select DM Sans and DM Mono.
2. Use the Download family option and extract the woff2 files.
3. Rename and copy the .woff2 files into this folder with the names above.

After placing real fonts, the app will load them from /fonts/fonts.css and work offline via the service worker cache.
